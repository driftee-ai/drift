
# Installation

There are several ways to install the `drift` CLI.


## Homebrew (macOS and Linux)

```bash
brew tap driftee-ai/drift
brew install drift
```

## Go

If you have a Go environment set up, you can install `drift` using `go install`:

```bash
go install github.com/driftee-ai/drift@latest
```

## Manual Installation

You can also download the latest release from the [GitHub Releases page](https://github.com/driftee-ai/drift/releases) and add it to your system's `PATH`.
